# Visitor Badge Generator

Create your own dynamic visitor counter badge and add it to your GitHub README in a few seconds.

Every time someone views the badge, the counter increases automatically.

---

## How to Generate Your Badge

1. Visit the badge generator:

   **https://stefanopaolonii.github.io/visitor-badge/**  

2. Customize:
   - Badge style  
   - Color  
   - Label color  
   - Label text  

3. Click **Generate Badge**

4. Copy the generated Markdown snippet

5. Paste it into your GitHub repository's `README.md`

Your badge will automatically update on every view.

---

## Features

- **Dynamic counter** that increases on every view
- **Fully customizable badge** (style, color, label)
- **Badge generated by this project** (not dependent on external services)
- Works in any **GitHub README** or documentation file

---

## Privacy Notice

This service does not track or store personal data.  
Only a numeric counter is stored for each badge.
